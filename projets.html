Projet 1 : Ergonomie d'une application Web

Ce projet consistait à moderniser et améliorer l'ergonomie d'une application Web destinée à la gestion des factures et des paramètres d'entreprise. Réalisé avec ASP.NET pour le backend et Angular pour le frontend, le projet comprenait :

  • Analyse de l'application existante.
  • Refonte de l'interface utilisateur pour une navigation intuitive.
  • Ajout de fonctionnalités interactives comme des tableaux dynamiques.
  • Tests approfondis pour garantir la compatibilité avec plusieurs navigateurs.

Projet 2 : Développement d'une application mobile

L'objectif de ce projet était de créer une application mobile multi-plateforme pour la gestion des tâches quotidiennes. Développé en Flutter, le projet incluait :

  • Conception d'une interface utilisateur minimaliste et responsive.
  • Implémentation de la gestion des tâches .
  • Intégration d'une base de données SQLite pour stocker les données localement.
